Event object - e

När du klickar på en ruta växlar den färg. Event-objektet skickas med som ett argument till funktionen och håller reda på vilken ruta som du klickade på.

var sektion = document.getElementsByClassName("conteiner"); sektion[0].addEventListener('click', function(e){ if (e.target.nodeName == "btnAll") { e.target.classList.toggle('ready'); } });
